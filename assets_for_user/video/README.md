# Куда класть видео первого запуска

1. Положи файл сюда и перемести/переименуй в **app/src/main/res/raw/intro_video.mp4**
2. Формат — mp4 (H.264 + AAC — самый совместимый вариант).
3. Видео проигрывается только один раз — при самом первом запуске приложения.
   После этого ставится флаг в настройках (DataStore), и видео больше не запускается,
   даже после перезапуска приложения.
4. Поверх видео автоматически появляется английская надпись:
   "Evest Antivirus destroyed virus Dusk."
   Текст уже зашит в strings.xml (first_run_message) — менять код не нужно.
5. Если файла intro_video.mp4 нет, приложение не упадёт — просто покажет
   надпись на чёрном фоне 2.5 секунды и пойдёт дальше (см. SplashVideoScreen.kt).

Совет: держи видео короче 10–15 секунд, чтобы первый запуск не казался долгим.
