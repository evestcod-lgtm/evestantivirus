# Куда класть аватар "Евестевень"

1. Удали плейсхолдер **app/src/main/res/drawable/avatar_evesteven.xml**
   (Android не даст собрать проект, если в одной папке будет и .xml, и .png с одинаковым именем).
2. Положи своё фото сюда: **app/src/main/res/drawable/avatar_evesteven.png**
   (можно .jpg — тогда и расширение в файле будет .jpg, конфликта не будет).
3. Больше ничего менять не нужно — MascotBubble.kt уже ссылается на
   R.drawable.avatar_evesteven, Android сам подхватит новый файл.

Аватар периодически всплывает на главном экране в виде фразы от Евестевеня,
например: «Я твой защитник Евестевень. Всё чисто, можешь быть спокоен.»
(логика — в HomeViewModel.kt, легко расширить список фраз).
