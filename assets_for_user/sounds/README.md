# Куда класть звуки

Положи файлы сюда и переименуй в **app/src/main/res/raw/** (папка `raw` создаётся, если её нет):

- notification.mp3
- virus.mp3
- error.mp3
- scan.mp3
- check.mp3

Важно для Android:
- Имена файлов только строчными латинскими буквами, цифрами и `_`, без пробелов и заглавных букв.
- Формат `.mp3` или `.ogg` — оба поддерживаются.
- Если какого-то файла нет — приложение не упадёт, просто не проиграет соответствующий звук (это заложено в SoundManager.kt).
